🔗 Chain of Thought (CoT)

Force le modèle à expliciter sa logique avant d'agir.

Q: "Meilleur resto italien >4.5★ acceptant CB ?"
T1: Je dois localiser l'utilisateur
A: get_location()
O: "Paris 11"
T2: Je cherche les restos italiens proches
A: search_restaurants(loc="Paris 11", cuisine="it")
O: [5 résultats]
T3: Filtrage par note & paiement
Final Answer: "Luigi's, 4.7★, 123 rue de la Roquette"
💡 Ajoutez "Pense étape par étape" dans le prompt pour activer le CoT automatiquement.

        if not human_approve(action): continue
    obs = run_tool(action)
    memory.add(thought, obs)
    if is_final(thought): return finalize(thought)

👤 Human-in-the-Loop (HITL)

🚦 Quand activer ?

  • Confiance < 0.7
  • Actions irréversibles
  • Données sensibles
  • Cas jamais vus

💡 Modes

  • Approval binaire
  • Correction directe
  • Guidance textuelle
  • Escalade expert

🎬 Scénario Réel

📞 Utilisateur : "Retour produit défectueux #78945, remboursement 1299€. Politique retour électronique ?"

🤖 T1: Vérif commande → 🔧 OrderStatus → ✅ Éligible retour, >100€
🤖 T2: Recherche politique → 🔧 KnowledgeBase → 📦 30j, emballage d'origine
🤖 T3: Traitement → 🔧 ProcessRefund → ⏳ En attente validation humaine
✅ Réponse: Ticket créé REF-5678. Instructions retour envoyées. Validation sous 24-48h.
👤 L'équipe support reçoit une notification, vérifie l'historique et approuve sous 2h. Tout est logué pour évaluation.
